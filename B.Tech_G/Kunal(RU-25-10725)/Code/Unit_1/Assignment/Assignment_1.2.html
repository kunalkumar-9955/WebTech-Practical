Purpose of the <a> Tag in HTML

The <a> tag in HTML is known as the anchor tag. It is used to create hyperlinks that connect one web page to another, or to a specific section within the same page. Hyperlinks allow users to navigate between documents, websites, or files easily.

Syntax:

<a href="URL" target="value">Link Text</a>

Example:

Visit OpenAI Website

Explanation of Each Part:

  • <a> — The opening anchor tag that starts the hyperlink.
  • href="URL" — The href (Hypertext Reference) attribute specifies the destination address or the link’s target URL (for example, another webpage, file, or section).
  • target="_blank" — This optional attribute defines how the link will open. The value _blank opens the link in a new browser tab or window.
  • Link Text — The clickable text that users see on the webpage. When clicked, it takes the user to the specified URL.
  • </a> — The closing anchor tag that ends the hyperlink.
