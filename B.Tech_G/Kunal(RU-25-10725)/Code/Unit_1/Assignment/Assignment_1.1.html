Understanding HTML Attributes

In HTML, attributes are used to provide additional information about HTML elements. They are always specified in the opening tag and usually come in name/value pairs like name="value". Attributes help control the behavior, appearance, and functionality of elements.

Example 1: The href Attribute (used in <a> tag)

The href attribute specifies the URL of the page the link goes to.

Example 2: The src and alt Attributes (used in <img> tag)

The src attribute defines the path of the image file, while the alt attribute provides alternate text if the image fails to load.



Explanation:

  • href — Defines the destination URL for a hyperlink.
  • src — Specifies the path (URL) of an image to display.
  • alt — Describes the image for accessibility and SEO.
