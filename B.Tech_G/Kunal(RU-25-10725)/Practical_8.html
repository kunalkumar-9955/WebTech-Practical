CSS Selectors Example

Class Selector (.classname)

This section is styled using a class selector. Class selectors can be reused for multiple elements.

Descendant Selector (Parent Child)

This paragraph is inside a div.box and styled using a descendant selector — it affects only paragraphs inside this box.

Group Selector (h1, h2, p)

Using a group selector we can apply the same style to multiple tags together — like h1, h2, and p.

Universal Selector (*)

The universal selector applies basic styling (font, margin reset) to all elements on the page.
