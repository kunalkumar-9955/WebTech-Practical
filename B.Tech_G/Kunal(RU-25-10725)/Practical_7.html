Using Box Model, Backgrounds, Borders, Margins, and Padding

About This Page

This webpage demonstrates the use of advanced CSS properties including the box model, background image, borders, margins, and padding. Each section is styled using external CSS to maintain a clean and modern layout.

1. Box Model Example

The box model includes content, padding, border, and margin. Here, the blue box has 20px padding, 3px solid border, and 20px margin around it.

2. Background Image Example

The page background uses a subtle image pattern combined with a solid color to enhance visual appeal.

3. Borders, Margins, and Padding

This box demonstrates rounded borders, spacing between sections (margins), and inner spacing (padding) that make content more readable and organized.
